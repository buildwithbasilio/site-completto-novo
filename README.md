# Completto MKT

Site institucional estático da Completto MKT, pronto para hospedagem no GitHub Pages.

## Arquivos principais

- `index.html`: estrutura e conteúdo do site.
- `styles.css`: identidade visual, responsividade, animações e layout.
- `script.js`: menu mobile, cursor, scroll reveal e interações.
- `ativos/completto-logo.png`: logo usada no site.
- `ativos/portfolio-lite/`: portfólio enxuto e otimizado para GitHub.
- `ativos/team/pdf/`: fotos otimizadas da equipe.

## Como hospedar no GitHub Pages

Envie todos os arquivos e a pasta `ativos` para o repositório. O arquivo `index.html` precisa ficar na raiz do projeto.

Para uma hospedagem leve, use apenas as pastas `ativos/portfolio-lite`, `ativos/team/pdf` e o arquivo `ativos/completto-logo.png`. Os materiais grandes antigos não são necessários para a versão publicada.
